It's time to let go

This site no longer supports Internet Explorer. Its dated architecture makes it a lot harder, and annoying, to develop experiences that work for it. Because of this, support for all versions of Internet Explorer has been dropped on this site.

Microsoft has announced that Internet Explorer 11, the latest version, will no longer be supported by them come August 2021. This means the browser will no longer receive updates and could potentially become insecure as exploits are discovered.

There are better browsers out there and you should seriously consider switching to one.

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Opera
